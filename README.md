# Having some good times with Angular2.

Requires NodeJS & npm.

To install:
```
– npm install
```

Run with:
```
– npm start
```

Build with:
```
– npm run build
```
*config/webpack.prod.js:*
When building remember to set *publicPath* (cdn) as the path or url containing your assets e.g scripts, images & css.
